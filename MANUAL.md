# CyberFly Keyboard 使用说明书

## 基本信息

| 项目 | 内容 |
|------|------|
| 设备名 | CyberFly KBD |
| 连接方式 | USB-C / 蓝牙 5.0 (BLE) |
| 蓝牙配对槽 | 6 个 (可连接 6 台设备) |
| 键盘布局 | 6 行 × 13 列, 78 键位 |
| MCU | nRF52840 (ARM Cortex-M4F) |
| 电池 | 锂电池供电, USB 充电 |

---

## 三层键位布局

### Layer 0: Default (默认层)

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐

├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│  E  │  Q  │  W  │ TAB │  R  │  T  │  Y  │  U  │  I  │  O  │  P  │  ]  │
```

### Layer 1: Fn (功能层, 按住 Fn 激活)

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
│     │     │     │     │     │     │     │ DEL │ F11 │     │BT清 │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│USB⇄ │ F1  │ F2  │ F3  │ F4  │ F5  │ F6  │ F7  │ F8  │ F9  │ F10 │ F12 │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │截图 │     │     │     │     │     │     │     │     │     │电源 │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│[Fn] │全选 │     │     │     │     │     │     │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │重置 │     │Boot │     │箭头 │     │     │     │PgUp │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │     │亮度 │亮度 │亮度 │扩展 │Home │PgDn │ End │关机 │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘

按住 Fn + 对应键:
  Fn + 1~0   → F1~F10        Fn + =   → F12
  Fn + -     → F11           Fn + DEL → Delete
  Fn + Q     → 截图 (Cmd+Shift+4)
  Fn + A     → 全选+复制 (Ctrl+A, Ctrl+C)
  Fn + C     → 清除所有设置并重启
  Fn + B     → 进入 UF2 Bootloader (刷机)
  Fn + M     → 输入 () => {}
  Fn + Space → 循环背光亮度
  Fn + RAlt  → 开关扩展层 (鼠标/蓝牙)
  Fn + RShft → 深度休眠 (关机)
  Fn + Enter → 清除当前蓝牙配对
  Fn + ]     → 开关外部电源
  Fn + ↑/↓   → Page Up / Page Down
  Fn + ←/→   → Home / End
  Fn + `     → 切换 USB/蓝牙输出
```

### Layer 2: Ext (扩展层, 按 Fn+RAlt 开关)

```
│退出 │BT 1 │BT 2 │BT 3 │BT 4 │BT 5 │BT 6 │     │     │     │BT清 │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│USB⇄ │     │     │     │     │     │     │     │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │     │     │     │滚↑  │     │鼠↑  │     │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │     │     │     │滚↓  │鼠←  │鼠↓  │鼠→  │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │     │     │     │滚←  │滚→  │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │     │中键 │左键 │ SPC │右键 │     │     │     │     │退出 │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘

进入/退出扩展层: Fn + RAlt (toggle)
扩展层内退出: ESC 或 RShift

蓝牙管理 (扩展层内):
  第一行彩色键 F1~F6 → BT 槽 1~6
  ESC 位置 → 退出扩展层
  Enter 位置 → 清除当前 BT 配对
  ` 位置 → 切换 USB/蓝牙

鼠标操作 (扩展层内):
  H/J/K/L → 鼠标 左/下/上/右
  Y/N → 滚轮 上/下
  N行左两键 → 滚轮 左/右
  左空格 → 左键
  中空格 → 空格 (保留)
  右空格 → 右键
  \ → 中键
```

## 快捷操作速查

### 蓝牙连接管理

| 操作 | 步骤 |
|------|------|
| 切换蓝牙槽 | Fn+RAlt 进扩展层 → 按第一行彩色键 1~6 → ESC 退出 |
| 切换 USB/蓝牙 | Fn + ` |
| 清除当前蓝牙配对 | Fn + Enter |
| 清除所有设置 | Fn + C (清除所有配对, 重启) |

### 首次蓝牙配对

1. 键盘开机 (插 USB 或电池供电)
2. Fn + RAlt 进入扩展层
3. 按第一行彩色键选择槽位 (如按 F1 位置 = 槽 1)
4. ESC 退出扩展层
5. 在手机/电脑蓝牙设置搜索 **"CyberFly KBD"** 并配对

### 切换已配对设备

1. Fn + RAlt 进入扩展层
2. 按对应槽位键 (1~6)
3. ESC 退出, 自动连接该设备

### 鼠标模式

| 操作 | 按键 (扩展层内) |
|------|----------------|
| 进入/退出 | Fn + RAlt |
| 移动光标 | H=左 J=下 K=上 L=右 |
| 左键 | 左空格 |
| 右键 | 右空格 |
| 中键 | \ |
| 滚轮上/下 | Y / N |
| 滚轮左/右 | 左侧两键 |

### 背光和电源

| 操作 | 按键 |
|------|------|
| 循环背光亮度 | Fn + Space (6 档 + 关) |
| 开关外部电源 | Fn + ] |
| 深度休眠 | Fn + RShift |

> 电池模式下背光自动关闭以节省电量, 仅 USB 供电时可用

### 系统功能

| 操作 | 按键 |
|------|------|
| 刷入新固件 | Fn + B → 拷贝 .uf2 到 NICENANO U盘 |
| macOS 截图 | Fn + Q |
| 全选+复制 | Fn + A |
| 箭头函数宏 | Fn + M → `() => {}` |
| Delete | Fn + Backspace |
| Page Up/Down | Fn + ↑/↓ |
| Home/End | Fn + ←/→ |

### 组合键 (同时按下)

| 按键 | 功能 |
|------|------|
| J + K | ESC |
| D + F | Tab |
| F + J | Enter |
| J + K + L | Caps Word |

## LED 指示灯

| 状态 | RGB LED 效果 |
|------|-------------|
| 开机 | 红→绿→蓝 依次闪烁 |
| USB 连接 | 红绿蓝持续循环 |
| 电池模式 | 每 5 秒蓝色双闪 |
| 扩展层开/关 | 绿色双闪 |

## 刷入固件

1. USB-C 连接键盘
2. 按 **Fn + B** 进入 Bootloader
3. 电脑出现 **NICENANO** U盘
4. 拷贝 `zmk.uf2` 到 U盘
5. 自动重启完成

---

## 故障排除

| 问题 | 解决 |
|------|------|
| 蓝牙搜不到 | Fn+` 确认在蓝牙模式 |
| 显示旧设备名 | iOS/macOS 缓存, 点连接后自动刷新 |
| iOS 不显示电量 | 添加"电池"小组件 |
| 按键无响应 | 可能在扩展层, Fn+RAlt 退出 |
| 恢复出厂 | Fn+C 清除所有设置 |
| Fn+ESC 无效 | 同列限制, 不影响使用 |

## 技术规格

| 参数 | 值 |
|------|-----|
| 蓝牙 | BLE 5.0, SoftDevice S140 v6.1.1 |
| USB | HID, VID 0x1209 / PID 0x0001 |
| 矩阵 | 6×13, ROW2COL, dome sheet (无二极管) |
| 背光 | AP3032 boost, 25kHz PWM, CIE gamma |
| HID | 复合设备: 键盘 + 鼠标 + 媒体键 |
| 续航 | ~10 天 (50mAh, BLE, 无背光) |
| 固件 | ZMK / Zephyr 4.1.0 |
