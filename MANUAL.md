# CyberFly 键盘使用说明书

CyberFly 是一款 BLE + USB 双模机械键盘，基于 nRF52840（Nice!Nano v2 引脚兼容模组）+ ZMK 固件。板载 **QMI8658A 6 轴 IMU** 支持空间飞鼠（开发中）。BLE 下 macOS / iOS 可读取电池电量（系统"我的设备"卡片、iOS 桌面电量小组件）。

- USB HID ID：VID `0x1209` / PID `0x0001`，设备名 `CyberFly Keyboard`（厂商 `CyberFly`）
- BLE device name：`CyberFly KBD`
- 最大同时配对数：**6** 台主机
- 矩阵：6 行 × 13 列（其中 C12 为飞线）；物理有效键位 **69 键**（空格按 1 键计），空格左/右区域计为独立键则为 **71 键**

## 指示灯一览


| 位置        | 颜色        | 含义                                  |
| --------- | --------- | ----------------------------------- |
| 键盘板**上方** | **RGB 灯** | 工作状态指示（开机动画、USB 循环、BLE 心跳、飞鼠模式切换提示） |
| RGB 旁边    | **红灯**    | **充电指示** —— 正在充电常亮，充满自动熄灭           |
| 键盘板**下方** | **白灯**    | **USB 5V 供电指示** —— 插 USB 线就亮        |


### RGB 工作状态详解（固件实现）


| 场景         | 效果                                   |
| ---------- | ------------------------------------ |
| 开机         | 红 → 绿 → 蓝 顺序闪一次然后熄灭（启动动画）            |
| USB 已连接    | **持续循环变换**（红绿蓝三色轮换）                  |
| 仅电池供电（BLE） | **蓝色双闪一次，每 5 秒一个心跳**，代表待机正常可随时使用     |
| 飞鼠模式切换     | OFF = 红闪、M1 = 绿闪、M2 = 蓝闪（双闪提示后回到原状态） |


## 工作模式


| 模式      | 连接      | 背光             | 飞鼠              |
| ------- | ------- | -------------- | --------------- |
| **USB** | USB-C 线 | ✅ 可用（Fn+空格调节）  | 可用              |
| **BLE** | 蓝牙      | ❌ **自动关闭**（省电） | **默认禁用**（需手动开启） |


> 背光 **仅 USB 通电时可用**，拔掉 USB 线后自动熄灭。配置上设了：`AUTO_OFF_USB=y`（USB 拔掉即灭）、`AUTO_OFF_IDLE=y`（空闲自动灭）。

- **Fn** 键在左侧第 4 行起始位（CapsLock 物理位置），按住激活 Fn 层
- **Esc** 双击激活 `caps_word`（一次性 caps，遇空格自动退出）
- **左 Shift** 双击锁 CapsLock
- **大空格键**：物理上是一整条长空格，在矩阵里占 3 列（左/中/右）；飞鼠模式开启后左右区域变成**鼠标左/右键**（smart_space 行为），中间保持空格

### Combo（组合键）快捷


| 同按              | 触发                      | 说明         |
| --------------- | ----------------------- | ---------- |
| **J + K**       | Esc                     | 50ms 同按判定  |
| **D + F**       | Tab                     | 50ms       |
| **J + K + L**   | CapsWord                | 50ms，一次性大写 |
| **F + J**       | Enter                   | 50ms       |
| **Fn + RShift** | **开/关飞鼠**（OFF→M1→M2 循环） | 75ms       |


### Macro（宏）


| 键      | 宏                       |
| ------ | ----------------------- |
| Fn + Q | 截屏（⌘ + Shift + 4）       |
| Fn + A | 全选 + 复制（⌘A → 50ms → ⌘C） |
| Fn + M | 箭头符号 `() => {}`         |


## Fn 层完整映射

按住 Fn（左侧 CapsLock 位）后键位变化：


| 原键          | Fn+ 后                                                  |
| ----------- | ------------------------------------------------------ |
| **B**       | **进入 Bootloader**（UF2 刷机模式）                            |
| **C**       | **Clean Settings**（清 NVS：BLE 配对、键盘缓存 → 自动 warm reboot） |
| 空格 × 3      | **循环切换背光** （`BL_CYCLE`）                                |
| Esc         | 切换输出 USB / BLE（OUT_TOG）                                |
| F1 ~ F6     | 切蓝牙设备 0 ~ 5（`BT_SEL 0..5`）                             |
| BSPC        | DEL                                                    |
| `-`         | F11                                                    |
| `=`         | F12                                                    |
| Enter       | **清除所有 BLE 配对**（BT_CLR）                                |
| 1 ~ 0       | F1 ~ F10                                               |
| Q           | 截屏宏                                                    |
| A           | 全选复制宏                                                  |
| ]           | 外部电源开关（EP_TOG）                                         |
| M           | `() => {}` 宏                                           |
| Up          | PgUp                                                   |
| Left        | Home                                                   |
| Down        | PgDn                                                   |
| Right       | End                                                    |
| LAlt / RAlt | **开/关飞鼠**（同组合键 Fn+RShift 效果）                           |
| RShift（单按）  | **软关机**（`soft_off`，比物理开关温和）                            |


> `Fn + B` 让芯片跳到 UF2 bootloader，会出现 `NICENANO` 虚拟 U 盘；`Fn + C` 只清 ZMK 设置并软重启（不擦固件）。
>
> `Fn + RShift` 在 75ms 内**同按**触发飞鼠切换；**单按** RShift（仍按住 Fn）超过 75ms 则触发软关机 `soft_off`。所以想关机要稍等一会儿再松手。

## 6 轴飞鼠（Air Mouse）

- **传感器**：QMI8658A（I2C 通信，同时带加速度计 + 陀螺仪）
- **驱动配置**：`CONFIG_QMI8658A=y` + `CONFIG_CYBERFLY_AIR_MOUSE=y`
- **ZMK Pointing**：`CONFIG_ZMK_POINTING=y`，走标准 HID Mouse Report

### 三档模式（循环切换）


| 模式      | 算法                        | 状态灯 |
| ------- | ------------------------- | --- |
| **OFF** | 关闭，IMU 进省电                | 红闪  |
| **M1**  | **Kalman 滤波 + 弹簧阻尼补偿**（稳） | 绿闪  |
| **M2**  | **加速度映射**（跟手）             | 蓝闪  |


**如何切换**：

- `Fn + RShift`（combo，75ms 内同按）
- `Fn + LAlt` / `Fn + RAlt`（单键也行，进入 Fn 层后按）

切换时 RGB 会对应颜色双闪一下。

### 飞鼠开启后中间 3 个空格键变化（Smart Space）


| 位置  | OFF 状态 | ON 状态（M1/M2） |
| --- | ------ | ------------ |
| 左空格 | 空格     | **鼠标左键**     |
| 中空格 | 空格     | 空格（保持）       |
| 右空格 | 空格     | **鼠标右键**     |


> ⚠️ 飞鼠固件仍在完善中（漂移滤波、加速度曲线正在调优）。稳定版发布前建议仅作体验使用。

## 蓝牙 / USB 切换

- **Fn + Esc** 切 USB / BLE 输出
- **Fn + F1 ~ F6** 切换蓝牙通道 0 ~ 5（支持同时保持 6 个主机配对）
- **Fn + Enter** `BT_CLR` 清除**当前通道**的配对记录
- **Fn + C** `reset_settings_bl` 清除**所有**设置（包括全部通道的配对）

### 配对流程

1. 新键盘首次开机自动广播
2. 在主机"蓝牙设置"里搜 `CyberFly KBD`，点配对
3. 连接后系统"我的设备"里显示电量（如图："已连接 - 🔋 100%"）
4. 想加第二台主机：`Fn + F2` → 配对 → ... 以此类推至 `Fn + F6`（最多 6 台）
5. 配对出问题："Fn + C" 清 NVS 重来

### 电量显示

- **macOS**: 系统设置 → 蓝牙 → "我的设备"卡片（见附图 "CyberFly KBD 已连接 - 🔋 100%"）
- **iOS / iPadOS**: 桌面加 "电池" 小组件
- 协议：标准 **BLE Battery Service (0x180F)** → **Battery Level (0x2A19)**

## 待机 / 续航

- BLE 待机 **> 48 小时**
- 长期不用：**拨电源开关到 OFF** 物理断电
- **配对信息不会丢失**（存 nRF52840 flash NVS，不靠电池保持）

## 电源 / 充电 / 开关

- 充电口：**USB-C**
- 充电指示：**旁边红灯** → 充电时常亮，**满电自动熄灭**
- USB 5V 供电指示：**板下白灯** → 插线就亮
- 硬件电源拨动开关：
  - **ON** — 电池供电，BLE 正常
  - **OFF** — 物理切断电池，**零功耗长期存放**

## Bootloader / 固件升级

### 进入 Bootloader 的两种方式

1. **Fn + B**（正常工作时）
2. **板载 reset 按钮双击**（固件损坏或 Fn+B 失效时）

进 bootloader 后：

- 板子主 LED 慢速呼吸（Adafruit UF2 0.10.0 指示）
- 出现名为 `**NICENANO`** 的 FAT16 虚拟 U 盘
- 把 `.uf2` 文件拖进去，写完自动重启到新固件

## 常见问题

**Q: 为什么背光不亮？**
A: 只有 USB 通电时才亮，BLE 模式下为了省电全关。Fn + 空格循环切换亮度。

**Q: 电量显示不准？**
A: 首次连接等 1 分钟左右让 BLE 电池服务同步；低于 20% 时 iOS 桌面小组件会红色图标。

**Q: Fn + B 后没出现 NICENANO 盘？**
A: USB 线可能只通电不走数据，换一根 data 线。

**Q: 我的 BLE 主机连不上？**
A: 按顺序试 1) `Fn + Enter` 清当前通道 → 2) `Fn + C` 清所有设置 → 3) 主机"忘记此设备"再配。

**Q: 按 Fn + RShift 有时出软关机有时出飞鼠？**
A: 75ms **同按**触发飞鼠切换（combo），**分前后按**会走 Fn 层 → 单按 RShift 的 `soft_off`。想关机按住 Fn 再慢点按 RShift；想切飞鼠就干脆同时按下。

**Q: 怎么从 OFF 状态快速回到 M2 模式？**
A: Fn+RShift 按两下（OFF→M1→M2）。

**Q: 长期不用怎么处理？**
A: 硬件开关拨 OFF，配对信息保留，下次回 ON 自动回连。
